# Инструменты для проверки и улучшения кода

ruff
- Быстрый линтер и форматер для поиска ошибок и проблем со стилем.
- Использование:
  ruff check .          # Проверить весь проект
  ruff check . --fix    # Автоматически исправить ошибки

black
- Автоформатер кода по стандарту PEP8.
- Использование:
  black --check .      # Проверить форматирование без изменений
  black .              # Отформатировать весь проект

monkeytype
- Сбор типов во время запуска и автоматическая генерация аннотаций.
- Использование:
  monkeytype run your_script.py    # Запуск с сбором типов
  monkeytype list-modules          # Список модулей с данными
  monkeytype apply your_module     # Вставить аннотации в код

no_implicit_optional
- Автоматически исправляет неявное использование Optional в аннотациях.
- Использование:
  no_implicit_optional fix path/to/code.py

# Рекомендации
- Сначала запускай monkeytype run для сбора типов.
- Затем применяй monkeytype apply для вставки аннотаций.
- Проверяй код через ruff и black, чтобы избежать ошибок и поддерживать стиль.
